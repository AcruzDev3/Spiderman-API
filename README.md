# 🕷️ Spiderman Crime API

Servicio REST desarrollado en **ASP.NET Core 9.0**, utilizando **Entity Framework Core**, **SQL Server** y autenticación mediante **JWT**.  
La API permite gestionar crímenes, villanos, ubicaciones y estados de casos dentro del universo de Spiderman, ofreciendo endpoints seguros, escalables y listos para integrarse con aplicaciones externas.

---

## 🚀 Características principales

- CRUD completo de **crímenes**.
- Gestión de **villanos**, **ubicaciones** y **estados**.
- Autenticación y autorización mediante **JSON Web Tokens (JWT)**.
- Persistencia con **Entity Framework Core** y **SQL Server**.
- Arquitectura limpia y preparada para escalar.
- Documentación automática con **Swagger / OpenAPI**.
- Manejo de excepciones y respuestas estandarizadas.

---

## 🧱 Tecnologías utilizadas

| Tecnología | Uso |
|-----------|-----|
| **ASP.NET Core 9.0** | Backend y controladores REST |
| **Entity Framework Core** | ORM y acceso a datos |
| **SQL Server** | Base de datos relacional |
| **JWT** | Autenticación y autorización |
| **Swagger** | Documentación interactiva |
| **C# 13** | Lenguaje principal |

---
## 🔐 Autenticación
La API utiliza **JWT Bearer Tokens**.
